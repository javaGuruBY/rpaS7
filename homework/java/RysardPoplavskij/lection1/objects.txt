1. Description
Задание : Описать объекты, с которыми вы сталкиваетесь каждый день.

Кол-во объектов - минимум 10.

****

1. Объект - Лампа настольная.
* Состояние:
    - Вкл/выкл
* Поведение:
    - включить - При включении освещает окружение
    - выключить - При выключении окружение не освещено

2. Объект - Увлажнитель.
* Состояние:
    - Вкл/выкл.
    - уровень воды.
* Поведение:
    - включить, При включении уменьшается количество воды в емкости.
    - выключить, количество воды в емкости неизменно.

3. Объект - Смарт часы.
* Состояние:
    - Вкл/выкл.
    - уровень заряда.
    - Замеряет пульс.
    - Показывает время.
    - Показывает количество пройденных часов.
* Поведение:
    - При однократном нажатии отображает время.
    - При двухкратном нажатии отображает количество шагом.
    - При трехкратном нажатии отображает пульс.
    - В зависимости от времени работы уменьшается уровень заряда.

4. Объект - корректор.
* Состояние:
    - Количество корректирующей ленты.
* Поведение:
    - Наносить корректирующий состав. При использовании количество корректирущей ленты уменьшается.

5. Объект - Наушники проводные.
* Состояние:
    - Подключены/отключены
* Поведение:
    - При подключении наушники воспроизводят звук, микрофон передает сигнал на подключенное оборудование.

6. Объект - Наушники беспроводные.
* Состояние:
    - вкл/выкл
    - уровень звука
    - режим поиска устройства
* Поведение:
    - включение при помощи нажатии кнопки и ее удержании в течение 3 секунд.
    - во включенном режиме, при нажатии кнопки увеличения звука, громкость увеличивается.
    - при удержании кнопки в течение восьми секунд начинается поиск оборудования.

7. Объект - Кружка.
* Состояние:
    - Количество жидкости.
* Поведение:
    - Хранить жидкость в зависимости от наливаемого и выливаемого.

8. Объект - Графин.
* Состояние:
    - Количество жидкости
* Поведение:
    - Хранить жидкость в зависимости от наливаемого и выливаемого.

9. Объект - Распылитель воды.
* Состояние:
    -количество жидкости
* Поведение:
    -в зависимости от использования количество воды уменьшается.

10. Объект - Зубная электрощетка
* Состояние:
    - Вкл/выкл
    - уровень заряда
    - Батарея разряжается/заряжается
* Поведение:
    - При включении работает электромотор
    - в зависимости от времени и количества раз использования уровень заряда уменьшается.
